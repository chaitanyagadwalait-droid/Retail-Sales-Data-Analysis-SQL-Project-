# 🛒 Retail Sales Data Analysis (SQL Project)

## 📌 Project Overview
This project focuses on analyzing retail sales data using SQL. The goal is to extract meaningful business insights such as customer behavior, sales trends, and category performance.

---

## 📂 Dataset Details
The dataset contains transactional retail data with the following columns:

- Transactions ID
- Sale Date & Time
- Customer ID
- Gender & Age
- Product Category
- Quantity Sold
- Price Per Unit
- Cost of Goods Sold (COGS)
- Total Sale

---

## 🛠️ Tools & Technologies
- SQL (PostgreSQL / MySQL)
- Data Cleaning
- Data Analysis

---

## 🔧 Data Cleaning Steps
- Checked for NULL values
- Removed incomplete records
- Renamed incorrect column (`quantiy` → `quantity`)

---

## 📊 Key Business Questions Solved

1. Total number of sales transactions
2. Number of unique customers
3. Category-wise sales performance
4. Average age of customers (Beauty category)
5. High-value transactions (sales > 1000)
6. Gender-wise transaction distribution
7. Monthly average sales & best-performing months
8. Top 5 customers by total sales
9. Unique customers per category
10. Sales distribution by time shifts (Morning, Afternoon, Evening)

---

## 📈 Key Insights
- Identified top-performing product categories
- Found peak sales months and seasonal trends
- Analyzed customer demographics and purchasing behavior
- Identified high-value customers for business targeting
